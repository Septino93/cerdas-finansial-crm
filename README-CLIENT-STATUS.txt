UPDATE CLIENT STATUS

Perubahan:
1. Tombol floating (+) dihapus karena tombol Tambah sudah tersedia di samping Daftar Client.
2. Tombol Hapus pada kartu client diganti menjadi Nonaktifkan.
3. Client nonaktif dapat diaktifkan kembali melalui tombol Aktifkan.
4. Data, konsultasi, pembayaran, dan riwayat client tidak dihapus.
5. Filter Aktif / Nonaktif sekarang menggunakan kolom is_active.

WAJIB SATU KALI:
Jalankan SETUP-CLIENT-STATUS.sql di Supabase SQL Editor sebelum menggunakan tombol Nonaktifkan.

Setelah itu upload seluruh isi folder ke GitHub untuk menggantikan versi sebelumnya.
