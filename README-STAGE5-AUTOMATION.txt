CERDAS FINANSIAL CRM - STAGE 5 AUTOMATION & COMMUNICATION

Fitur:
1. Communication Center pada setiap kartu konsultasi.
2. Template WhatsApp: booking, pembayaran, jadwal, reminder H-1, dan terima kasih.
3. Data client, layanan, nomor konsultasi, nominal, jadwal, metode, dan link terisi otomatis.
4. Template Manager melalui Profil > Template Komunikasi.
5. Riwayat komunikasi dicatat ke activity_logs ketika WhatsApp dibuka.
6. Follow up dapat dijadwalkan dan dicatat ke activity_logs.
7. Dashboard menampilkan jumlah reminder H-1 untuk konsultasi besok.

CATATAN:
- Sistem menggunakan wa.me; pesan tidak dikirim otomatis. Admin tetap menekan tombol kirim di WhatsApp.
- Template disimpan pada localStorage browser admin. Tidak memerlukan SQL tambahan.
- Tidak menggunakan WhatsApp Business API dan tidak menambah biaya layanan.
